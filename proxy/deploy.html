Studio Cinema

Deploie ton proxy en 5 minutes pour generer tes videos avec Veo.

Avant de commencer : tu dois avoir un compte Google Cloud avec la facturation activee et une cle API Vertex AI Express Mode. Les credits Google Cloud couvrent l utilisation.

1
Clique sur Ouvrir Cloud Shell ci-dessous et connecte-toi avec ton compte Google
2
Copie la commande ci-dessous et colle-la dans le terminal avec Ctrl+V puis Entree
3
Attends 3 a 5 minutes que le deploiement se termine
4
A la fin tu verras URL PROXY s afficher — copie-la et colle-la dans Studio Cinema
Commande a copier-coller
gcloud services enable run.googleapis.com cloudbuild.googleapis.com storage.googleapis.com aiplatform.googleapis.com --quiet && mkdir -p ~/proxy && cd ~/proxy && curl -fsSL https://raw.githubusercontent.com/studioirl/studio_irl/main/proxy/index.js -o index.js && curl -fsSL https://raw.githubusercontent.com/studioirl/studio_irl/main/proxy/package.json -o package.json && gcloud run deploy studio-cinema-proxy --source . --region europe-west1 --allow-unauthenticated --memory 512Mi --quiet && PROJECT=$(gcloud config get-value project) && BUCKET="studio-cinema-${PROJECT}" && gsutil mb -l europe-west1 gs://${BUCKET} 2>/dev/null || true && gsutil iam ch allUsers:objectViewer gs://${BUCKET} && echo "" && echo "=============================" && echo "=== DEPLOIEMENT TERMINE ===" && echo "=============================" && echo "" && echo "=== URL PROXY (a copier dans Studio Cinema) ===" && gcloud run services describe studio-cinema-proxy --region europe-west1 --format="value(status.url)" && echo "" && echo "=== BUCKET GCS ===" && echo "gs://${BUCKET}" && echo ""
Ouvrir Cloud Shell
A la fin du deploiement tu verras

=== URL PROXY (a copier dans Studio Cinema) ===
https://studio-cinema-proxy-XXXXXXXXXX.europe-west1.run.app

Copie cette URL et colle-la dans le champ URL de mon proxy sur l ecran d accueil de Studio Cinema.

Chaque eleve deploie son propre proxy — les frais vont sur ton propre compte Google Cloud.
