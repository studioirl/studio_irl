Studio Cinema

Deploie ton proxy en 5 minutes pour generer tes videos avec Veo.

1
Clique sur Ouvrir Cloud Shell ci-dessous et connecte-toi avec ton compte Google Cloud
2
Copie la commande ci-dessous et colle-la dans le terminal avec Ctrl+V puis Entree
3
Attends 3 minutes que le deploiement se termine
4
Copie l URL du service qui s affiche a la fin et colle-la dans Studio Cinema
Commande a copier-coller
gcloud services enable run.googleapis.com cloudbuild.googleapis.com storage.googleapis.com aiplatform.googleapis.com --quiet && mkdir -p ~/proxy && cd ~/proxy && curl -fsSL https://raw.githubusercontent.com/studioirl/studio_irl/main/proxy/index.js -o index.js && curl -fsSL https://raw.githubusercontent.com/studioirl/studio_irl/main/proxy/package.json -o package.json && gcloud run deploy studio-cinema-proxy --source . --region europe-west1 --allow-unauthenticated --memory 512Mi --quiet
Ouvrir Cloud Shell
